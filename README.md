# Python企业员工管理系统 python807

## 项目概述

企业员工管理系统是一套全面的管理解决方案，旨在通过高效的信息管理提高企业内部工作效率。系统涵盖了部门、用户、任务和订单等多个管理模块，实现了企业员工信息和工作流程的有效整合。

## 技术栈

- **后端**: Python, Django
- **前端**: HTML, CSS, JavaScript, jQuery
- **数据库**: MySQL
- **开发工具**: Vscode, navicat

## 运行环境

- Python 3.0及以上版本
- MySQL 8.0

## 功能模块

- **部门管理**: 对企业内部门结构进行维护和管理。
- **用户管理**: 管理企业员工信息，包括权限分配等。
- **订单管理**: 处理订单相关信息，确保订单流程的高效运转。
- **任务管理**: 发布、追踪和管理员工任务。
- **手机管理**: 集中管理企业配发的手机设备。
- **设备管理**: 对企业各类设备进行管理。
- **管理员管理**: 系统管理员的相关操作管理。
- **订单数据可视化**: 以图表形式直观展示订单数据。
- **修改个人信息**: 员工可自主更新个人信息。

## 系统角色

- 系统管理员
- 普通员工

## 部署步骤

1. 确保Python环境和MySQL数据库已正确安装。
2. 克隆或下载项目代码。
3. 安装项目依赖：`pip install -r requirements.txt`。
4. 导入数据库结构：使用navicat等工具导入sql文件。
5. 修改配置文件中的数据库连接信息。
6. 运行Django开发服务器：`python manage.py runserver`。
